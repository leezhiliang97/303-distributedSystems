# Chapter 4 - Interprocess Communication

## Java UDP Client/Server

### Pre-requisites

1. Eclipse Java

2. Git

### Compiling

```
$ javac UDPServer.java
$ javac UDPClient.java
```

### Running

```
$ java UDPServer
$ java UDPClient
```
